# AI Clipart Generator (Android)

## Tech Stack
- Expo + React Native
- Supabase Edge Functions (secure proxy)
- Hugging Face Inference (free tier)
- Stitch → manually refined UI
- NativeWind + Reanimated + Zustand

## Tradeoffs
- Used text-to-image with detailed prompt (fast & reliable for demo)
- Parallel generation for all 5 styles
- PNG download + native share
